# Imgur Extension

[Website/Releases](http://www.polydes.com/repo/toolset/com.polydes.imgur) |
[Issues](https://github.com/polydes/imgur/issues)

</div>

This [Stencyl](https://www.stencyl.com/) extension allows you to upload screenshots to [imgur](https://imgur.com/) directly out of Stencyl (using the camera button in Design Mode or Scene Designer).

## Using this Stencyl extension

This is a free extension. Installation instructions and guides can be found [here](http://www.polydes.com/repo/toolset/com.polydes.imgur/).
